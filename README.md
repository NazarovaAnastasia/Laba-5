Описание проекта
Проект представляет собой простой веб-приложение для учета личных финансов. Пользователи могут добавлять доходы и расходы, классифицировать их по категориям, просматривать статистику за различные периоды и устанавливать бюджетные цели. Приложение имеет минималистичный интерфейс и работает в браузере без необходимости установки дополнительного программного обеспечения. Основная цель — помочь пользователям контролировать свои финансы через наглядную визуализацию данных.
